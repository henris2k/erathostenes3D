# 🌍 Eratóstenes en 3D

Visualización interactiva del método que usó Eratóstenes para calcular la circunferencia de la Tierra hace más de 2.000 años.

## 📐 Qué muestra

- Una esfera 3D (la Tierra)
- Dos puntos: Siena (Asuán) y Alejandría
- El arco del círculo máximo entre ellos (la distancia real)
- Los radios desde el centro de la Tierra
- El ángulo central que Eratóstenes midió con la sombra

pip install numpy plotly haversine
